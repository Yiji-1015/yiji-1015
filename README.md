## Repo Description

**1) matching-agent**  
석사학위논문 연구를 기반으로, RAG와 역할 분리형 LLM 구조를 결합한 Matching Agent 구현 프로젝트입니다.  
Langgraph를 활용하여, 의도·맥락·역할 상보성 등을 고려한 의미 기반 매칭을 다중 LLM 에이전트 구조로 설계했습니다.

**2) upthink**  
Upstage Solar Pro 2를 활용해 Obsidian 기반 개인 지식 관리 자동화를 구현한 협업 프로젝트입니다. (업스테이지 엠버서더 1기 활동의 일환)
PM으로서 기능 기획·아키텍처 설계·개발에 참여했으며, 태그 추천·연관 노트 검색·노트 분할 등 핵심 기능을 구현했습니다.

**3) hongdae_restaurants**
홍대 음식점 및 리뷰 데이터를 활용해 FAISS와 BM25를 결합한 RAG 하이브리드 검색 기반 추천 챗봇입니다.  
리뷰 임베딩을 적용해 검색 문맥 이해도를 높이고, MongoDB와 Streamlit으로 서비스 환경을 구성했습니다.

**4) outreach-support-tool**  
공개 웹 데이터를 크롤링해 유권자 특성을 구조화하고, 개인화 메시지 생성을 실험한 AI 프로젝트입니다.  
프로필 정보 요약과 프롬프트 설계를 통해 맞춤형 메시지 생성 파이프라인을 구현했습니다.

**5) kr-stock-data-study**  
한국 주식시장(KOSPI·KOSDAQ)과 주요 경제지표, 서울 부동산 매매가 간의 관계를 데이터 기반으로 분석한 프로젝트입니다.  
상관분석과 회귀분석을 통해 자산 시장 간 구조적 연관성을 통계적으로 검증했습니다.

**6) freight-volume-dashboard**  
물류회사 컨설팅팀 인턴 과제로, 납품처 물동량 데이터를 주소→좌표로 정규화하고 지역 단위로 집계·시각화한 대시보드 프로젝트입니다.  
Oracle 기반 데이터 처리와 지오코딩/역지오코딩을 통해 Folium 지도(히트맵·마커)로 물동량 분포를 표현했습니다.

**7) guide-demo**  
RAG, 머신러닝, 오픈소스 활용 등을 개인 학습 목적으로 자유롭게 실험한 코드와 데모를 모아둔 레포지토리입니다.  
기능 단위 구현과 아이디어 검증을 중심으로 구성했습니다.
